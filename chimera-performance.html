Performance

There aren’t published Chimera benchmarks yet. The design is built for hardware-rate throughput — asynchronous throughout, zero-copy through libevpl, capability-negotiated backends — but the work to publish reproducible numbers across realistic configurations is still in progress.

Benchmarks aren’t published yet.

The plan is to publish reproducible benchmarks here: NFS, SMB, and S3 throughput and latency against the reference backends, with the methodology, hardware details, and the harnesses in the repository.

Until then, the nearest available numbers are one layer down. See libevpl performance for the runtime Chimera runs on.
